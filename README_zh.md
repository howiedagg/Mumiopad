# Mumiopad - 極簡無線觸控板與虛擬鍵盤

Mumiopad 是一款輕量且安全的開源工具，能將您的 Android 手機變身為 Windows 電腦的無線觸控板與虛擬鍵盤。適合您在沙發上放鬆、觀賞影音、簡報，或是需要輕量遠端控制電腦的日常場景。

透過高效率的本機區域網路連線，提供流暢的游標追蹤與低延遲輸入，且完全不需要手動配置繁瑣的 IP 位址。

---

## 📥 快速連線指引

請至 👉 **[Mumiopad 最新版本下載頁面](https://github.com/howiedagg/Mumiopad/issues)**

### 1. 電腦端（Windows）

1. 下載 `MumiopadServer-vx.x.x.exe`
2. 直接雙擊執行
3. 程式會常駐於右下角系統匣，默默在背景服務，不佔用桌面空間

### 2. 手機端（Android）與配對

1. 下載 `Mumiopad-vx.x.x.apk` 並完成安裝
2. 確保手機與電腦連線至同一個 Wi-Fi 區域網路
3. 開啟 App，自動搜尋電腦
4. 點擊您的電腦名稱
5. 電腦畫面出現配對視窗後按下「是」即可完成連線

## 🎮 觸控板手勢與功能

我們設計了符合直覺的操作手勢，並全面對接 Android 系統級觸覺震動，提供踏實的盲操反饋：

| 手勢指數 | 操作行為 | 模擬滑鼠與鍵盤功能 |
|----------|----------|-------------------|
| **單指** | 滑動 | 移動游標 |
| | 輕點一下 | 滑鼠左鍵 |
| | 按住 200 毫秒拖曳 | 選取／拖曳（按住時鎖定震動，放開時釋放震動） |
| **雙指** | 同時輕點 | 滑鼠右鍵 |
| | 上下滑動 | 網頁滾動（細密滾動震動） |
| | 左右刷動 | 瀏覽器上一頁／下一頁 |
| | 雙指捏合 | Windows 螢幕放大鏡（高精度刻度震動） |
| **三指** | 輕點一下 | 多工作業檢視 (`Win + Tab`) |
| | 同時上/下滑 | **顯示/還原桌面** (`Win + D`) |
| | 同時左/右滑 | **切換/開關手機虛擬鍵盤** |
| **四指** | 輕點一下 | 滑鼠中鍵 |
| **實體鍵** | 音量鍵 | 同步控制電腦系統音量 |

## ⚙️ 現代極簡選單管理

點擊手機狀態列膠囊即可開啟管理面板。

### 燈號語意

- 🟢 綠燈：目前已連線
- 🔵 藍燈：已配對且在線
- ⚪ 灰燈：已配對但離線
- 🟡 橘燈：新發現尚未配對

### 手勢操作

- 短點：切換連線
- 長按綠色列：中斷連線
- 左滑列項：解除配對

### 手動重新整理 🔄

- 每次開啟選單自動掃描 5 秒
- 掃描結束後顯示 🔄 圖示
- 點擊即可重新搜尋附近裝置

## 🛠️ 開發者建置說明（Build from Source）

### 1. 電腦伺服器端（Python 3.10+）

    cd pc-server-python
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    python server.py

打包 EXE：

    pyinstaller --onefile --noconsole --name MumiopadServer server.py

### 2. Android App（Kotlin / Compose）

使用 Android Studio 開啟 `android-app`

建置：

    cd android-app
    ./gradlew assembleDebug

APK 輸出位置：

    android-app/app/build/outputs/apk/debug/app-debug.apk
