# Mumiopad - 極簡雙模無線觸控板

控制電腦，躺著也能搞定。

🌐 **[English](README.md)**

**Mumiopad** 是一款極簡的雙模無線觸控板與虛擬鍵盤。支援 **Wi-Fi 區域網路**與**免驅動藍牙（Bluetooth HID）**，讓您的 Android 手機化身為滑鼠與鍵盤，無論是在沙發放鬆、觀影、簡報，遠端控制皆輕鬆自如。

👉 **[下載最新版本 (APK / EXE)](https://github.com/howiedagg/Mumiopad/releases)**

---

## 📥 快速連線

### 📶 Wi-Fi 模式
1. **電腦**：啟動 `MumiopadServer.exe`。
2. **手機**：連上相同 Wi-Fi 並開啟 App。
3. **連線**：點擊電腦名稱，並在電腦彈窗點「是」即可。

### ᛒ 藍牙模式 (免裝驅動)
1. **手機**：開啟 App 切到藍牙，點擊「開放藍牙配對」。
2. **電腦**：在 Windows 藍牙設定中搜尋並配對**「您的手機名稱」**。
3. **連線**：配對成功後，在手機清單點擊電腦即可控制。

## 💡 智慧連線

* **地點感知**：自動綁定不同 Wi-Fi 下的預設電腦。
* **一鍵雙模**：自由切換 Wi-Fi 或藍牙。
* **多機隨切**：支援配對多台電腦，於選單中點選即可無縫切換。

## 🛠️ 開發建置

### 1. 電腦伺服器端（Python 3.10+）

    cd pc-server-python
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    python main.py

打包 EXE：

    pyinstaller --onefile --noconsole --name MumiopadServer main.py

### 2. Android App（Kotlin / Compose）

使用 Android Studio 開啟 `android-app`

建置 APK：

    cd android-app
    ./gradlew assembleDebug
