# Mumiopad - 手機無線觸控板與虛擬鍵盤

Mumiopad 是一個可以將 Android 手機輕鬆變成電腦無線觸控板與鍵盤的工具。適合您在沙發上放鬆、觀賞影音、簡報，或是需要輕量遠端控制電腦的日常場景。

透過高效率的本機端連線，提供極流暢的游標追蹤與低延遲的輸入模擬，且完全不需要繁瑣的網路 IP 設定。

---

## 📥 快速開始（如何下載與連線）

請先點擊前往 👉 **[Mumiopad 最新版本下載頁面](https://github.com/howiedagg/Mumiopad/issues)** 取得以下檔案：

### 1. 電腦端設定（Windows）
*   在下載頁面中，點擊並下載 `MumiopadServer-vx.x.x.exe`。
*   **直接雙擊執行**下載好的 `.exe` 檔案。
*   執行後，程式會直接常駐於畫面右下角系統匣，並在背景默默為您服務，不佔用任何桌面空間。

### 2. 手機端設定（Android）
*   在下載頁面中，點擊並下載 `Mumiopad-vx.x.x.apk` 到您的手機並完成安裝。
*   確保手機與電腦連線至 **同一個 Wi-Fi 網路**。
*   打開手機 App，它會自動搜尋附近的電腦。點擊您的電腦名稱，電腦螢幕便會彈出配對視窗，點擊「是」即可完成連線！

## 🎮 觸控板手勢與功能介紹

我們設計了符合日常直覺的操作方式，並加入觸覺震動來輔助無視覺盲操：

*   **單指操作**：
    *   `移動游標`：在觸控板區域滑動。
    *   `滑鼠左鍵`：單指輕點一下。
    *   `長按並拖曳`：手指按住觸控板不動約 200 毫秒，手機會發出一次點擊震動，此時即可開始拖曳視窗或選取文字。手指抬起時會伴隨釋放震動。
*   **雙指操作**：
    *   `滑鼠右鍵`：雙指同時輕點。
    *   `滾動網頁`：雙指同時上下滑動（可在設定中調整滾動方向與速度）。
    *   `瀏覽器上一頁/下一頁`：在觸控板任何地方，雙指左右刷動。
*   **三指操作**：
    *   `多工作業檢視 (Win+Tab)`：三指輕點一下。
    *   `顯示桌面`：三指同時向下垂直滑動。
    *   `還原所有視窗`：三指同時向上垂直滑動。
*   **四指操作**：
    *   `開關虛擬鍵盤`：四指同時在觸控板輕點，即可快速叫出或收合手機鍵盤，直接在手機上輸入中英文。
*   **手機實體音量鍵**：
    *   在連線成功時，按手機側邊的實體音量鍵，即可直接調整電腦系統的主音量（未連線時自動恢復為調整手機音量）。

## ⚙️ 系統設定調整
您可以隨時點擊手機 App 右上角的齒輪圖示，進行以下微調：
*   **滑鼠移動速度**（支援 0.3x ~ 3.0x 靈敏度調整）。
*   **滾動速度**。
*   **反向滾動（自然滾動）** 開關。

## 🛠️ 從原始碼自行建置 (Build from Source)

如果您是開發者，或想自行修改並編譯本專案，可以參考以下步驟：

### 1. PC 伺服器端 (Python)
確保電腦已安裝 Python 3.10+ 環境：
```bash
cd pc-server-python
python -m venv .venv
source .venv/bin/activate  # Windows 請執行 .venv\Scripts\activate
pip install -r requirements.txt
python server.py
```
若想自行打包成單一無主控台的 `.exe` 執行檔，可在虛擬環境中執行：
```bash
pyinstaller --onefile --noconsole --name MumiopadServer server.py
```

### 2. Android App 端 (Kotlin/Compose)
1. 使用 **Android Studio** 開啟 `android-app` 資料夾。
2. 確保環境已配置 JDK 17。
3. 在終端機執行以下指令建置 APK：
```bash
cd android-app
./gradlew assembleDebug
```
編譯完成的 APK 檔案將會生成於 `android-app/app/build/outputs/apk/debug/app-debug.apk`。
